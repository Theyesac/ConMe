ConMe README
============

Current Tasks
-------------

* Account login system
  - Work has began, still needs sessions, password storage, verification, etc.
* User profile system
  - First/Last name, Nickname, Location, Multiple Photos, Links to arbitrary social networks, Convention & Cosplay History.
  - User search with cosplay and convention search. (So you can find the person who cosplayed Ash from Pokemon at Ohayocon for example.)

Getting Started
---------------

- cd <directory containing this file>

- $venv/bin/python setup.py develop

- $venv/bin/populate_ConMe development.ini

- $venv/bin/pserve development.ini


Production Launch Checklist
---------------------------

- Update Session key & secret
